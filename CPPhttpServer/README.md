# CPP HTTP Server

## This program is a webserver used to host local files over your current IPv4 or localhost in your working directory. No setup is required, just run the program and it is running.

## Either download the binaries and extract them and run them or compile the source yourself.

## The used HTTP library used can be found at https://github.com/yhirose/cpp-httplib/blob/master/httplib.h
